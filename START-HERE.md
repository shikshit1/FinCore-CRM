```bash

```

### Step 4: Configure (2 minutes)
Create `server/.env`:
```
MONGODB_URI=mongodb+srv://...
JWT_SECRET=your_secret_key
```

Create `client/.env.local`:
```
VITE_API_URL=http://localhost:5000/api
```
